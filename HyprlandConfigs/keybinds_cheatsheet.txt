# HYPRLAND KEYBINDS CHEATSHEET
# -------------------------------------------------------------
# Modifier: SUPER (Windows Key)
# -------------------------------------------------------------

### LAUNCHERS & APPLICATIONS
SUPER + A   -> Application Launcher (Rofi)
SUPER + Q   -> Launch Terminal (Kitty)
SUPER + E   -> Launch File Manager (Nautilus)
SUPER + B   -> Launch Chrome
SUPER + C   -> Launch Gnome Calculator
SUPER + W   -> Launch Work Folder (~/Documents/work)

### SYSTEM ACTIONS & CONFIGURATION
SUPER + F1          -> Show Keybinds Cheatsheet (Rofi)
SUPER + SHIFT + R   -> Reload Hyprland Configuration
SUPER + M           -> Exit Hyprland

### WINDOW MANAGEMENT (Dwindle Layout)
SUPER + K   -> Kill Active Window
SUPER + V   -> Toggle Floating Mode
SUPER + P   -> Toggle Pseudotiling
SUPER + J   -> Toggle Split

# FOCUS & MOVEMENT
SUPER + Left    -> Move Focus Left
SUPER + Right   -> Move Focus Right
SUPER + Up      -> Move Focus Up
SUPER + Down    -> Move Focus Down
SUPER + LMB Drag -> Move Window
SUPER + RMB Drag -> Resize Window

### WORKSPACE MANAGEMENT
SUPER + [1-0]       -> Switch to Workspace 1-10
SUPER + SHIFT + [1-0] -> Move Active Window to Workspace 1-10
SUPER + Scroll Up   -> Next Workspace
SUPER + Scroll Down -> Previous Workspace
SUPER + S           -> Toggle Special Workspace (Scratchpad: magic)
SUPER + SHIFT + S   -> Move Active Window to Special Workspace

### SCREENSHOTS (grim/slurp)
Print Screen      -> Screenshot Area (Save & Notify)
SHIFT + Print Screen -> Screenshot Full Screen (Save & Notify)
SUPER + SHIFT + S -> Screenshot Area (Save & Notify - Secondary Bind)
SUPER + SHIFT + Print Screen -> Screenshot Area to GIMP

### MULTIMEDIA KEYS (Requires `wpctl`, `brightnessctl`, `playerctl`)
XF86AudioRaiseVolume -> Volume Up (5%)
XF86AudioLowerVolume -> Volume Down (5%)
XF86AudioMute      -> Mute/Unmute Volume
XF86AudioMicMute   -> Mute/Unmute Microphone
XF86AudioNext      -> Media Next Track
XF86AudioPrev      -> Media Previous Track
XF86AudioPause/Play -> Media Play/Pause
XF86MonBrightnessUp -> Brightness Up (5%)
XF86MonBrightnessDown -> Brightness Down (5%)
